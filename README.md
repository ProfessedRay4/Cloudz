# auto raid-discord

### Installation:
1) Download zip (or git clone)
2) extract zip
3) ``pip install -r requirements.txt``
4) goto [here](https://discord.com/developers/) and get bot token
5) edit `config.json`
6) run setup.bat OR `python main.py`
7) enjoy!
